# @dhruv2mars/gunmetal

Install Gunmetal and run the local AI switchboard.

## Install

The package name is reserved here, but not published yet.

Today, use the repo locally or a release asset.

Planned package:

```bash
npm i -g @dhruv2mars/gunmetal
```

First run downloads the native `gunmetal` binary into `~/.gunmetal/bin/`.

## Quickstart

```bash
gunmetal setup
gunmetal start
```

Then point your app at:

- base URL: `http://127.0.0.1:4684/v1`
- API key: the Gunmetal key created during setup

## Commands

```bash
gunmetal
gunmetal setup
gunmetal start
gunmetal status
gunmetal profiles list
gunmetal keys list
gunmetal logs list
```
