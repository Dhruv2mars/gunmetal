# @gunmetal/web

Marketing site and product docs for Gunmetal.

## Local dev

```bash
bun run dev
```

Open `http://localhost:3000`.

## Structure

- `src/app/` route entrypoints
- `src/components/` shared UI
- `src/lib/` page content and helpers

## Checks

```bash
bun run test   # typecheck
bun run lint   # eslint
bun run build  # production build
```

Read `AGENTS.md` in this folder before changing Next.js code.
